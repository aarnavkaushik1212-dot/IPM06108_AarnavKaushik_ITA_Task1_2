Student Performance Prediction (ML Project)

Overview

This project analyzes student data and predicts whether a student will Pass or Fail using machine learning. It also identifies learning behavior patterns using clustering.

Techniques Used
	•	Exploratory Data Analysis (EDA): Correlation heatmap to understand relationships between student factors.
	•	K-Means Clustering: Groups students into learning profiles:
	•	Irregular Learners
	•	Average Learners
	•	Consistent Learners
	•	Supervised Learning: Predicts Pass/Fail using:
	•	Logistic Regression
	•	Random Forest

Features Used
	•	studytime
	•	attendance
	•	absences
	•	failures
	•	G1 (First grade)
	•	G2 (Second grade)
	•	G3 (Final grade)

Evaluation Metrics
	•	Cross-validation accuracy
	•	Precision
	•	Recall
	•	F1-score

Outcome

The model helps identify student performance patterns and predicts academic outcomes to support better educational decisions.
